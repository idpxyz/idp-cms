## 🔄 工作流程

### 场景 1：houad 创建专题文章

```
1. houad 登录后台
   └─→ 只能看到 2 个频道：教育健康、视频专题

2. houad 在"视频专题"频道创建文章
   └─→ 编辑内容、上传图片

3. houad 保存文章
   └─→ 文章处于草稿状态
   └─→ ❌ 看不到"发布"按钮

4. chensiyan 或 leo5122 审核
   └─→ 检查内容、完善格式

5. chensiyan 或 leo5122 发布
   └─→ ✅ 文章正式上线
```

### 场景 2：chensiyan 直接发布文章

```
1. chensiyan 登录后台
   └─→ 可以看到所有 12 个频道

2. chensiyan 在任意频道创建文章
   └─→ 编辑内容

3. chensiyan 直接发布
   └─→ ✅ 文章立即上线（无需其他人审核）
```
