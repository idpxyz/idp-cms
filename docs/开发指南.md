# IDP-CMS AI 门户网站开发指南

## 项目概述

本项目基于 IDP-CMS 架构，构建了一个专业的 AI 门户网站（aivoya.com），提供 AI 工具导航、行业资讯聚合和技术教程等功能。项目采用现代化的技术栈，包括 Wagtail CMS 作为内容管理后端，Next.js 作为前端框架。

### 技术架构

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Next.js 15    │    │   Wagtail 7.1   │    │   OpenSearch    │
│   (前端门户)      │◄──►│   (内容管理)      │◄──►│   (搜索引擎)      │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                        │                        │
         │                        │                        │
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Tailwind CSS  │    │   PostgreSQL    │    │   ClickHouse    │
│   (样式框架)      │    │   (主数据库)      │    │   (分析数据库)    │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 核心功能

- **AI 工具导航**: 分类展示各类 AI 工具，支持搜索和筛选
- **AI 资讯聚合**: 实时聚合 AI 行业动态和技术突破
- **技术教程**: 提供 AI 相关的技术教程和学习资源
- **内容管理**: 基于 Wagtail 的强大内容管理系统
- **搜索功能**: 基于 OpenSearch 的全文搜索和推荐系统

```python
from rest_framework.decorators import api_view
from rest_framework.response import Response
from django.db.models import Q
from apps.core.models import Channel, Region

@api_view(["GET"])
def ai_tools(request):
    """获取AI工具列表"""
    # 获取查询参数
    category = request.query_params.get("category")
    search = request.query_params.get("search")
    pricing = request.query_params.get("pricing")
    is_hot = request.query_params.get("is_hot")
    is_new = request.query_params.get("is_new")
    page = int(request.query_params.get("page", 1))
    size = int(request.query_params.get("size", 20))

    # 构建查询
    queryset = AIToolPage.objects.live().public()

    if category:
        queryset = queryset.filter(category=category)

    if search:
        queryset = queryset.filter(
            Q(title__icontains=search) |
            Q(description__icontains=search) |
            Q(tags__name__icontains=search)
        ).distinct()

    if pricing:
        queryset = queryset.filter(pricing=pricing)

    if is_hot == "true":
        queryset = queryset.filter(is_hot=True)

    if is_new == "true":
        queryset = queryset.filter(is_new=True)

    # 分页
    start = (page - 1) * size
    end = start + size
    tools = queryset[start:end]

    # 序列化数据
    tools_data = []
    for tool in tools:
        tools_data.append({
            "id": tool.id,
            "title": tool.title,
            "description": tool.description,
            "tool_url": tool.tool_url,
            "logo_url": tool.logo_url,
            "category": tool.category,
            "pricing": tool.pricing,
            "features": tool.features,
            "rating": float(tool.rating),
            "usage_count": tool.usage_count,
            "is_hot": tool.is_hot,
            "is_new": tool.is_new,
            "tags": [tag.name for tag in tool.tags.all()],
            "url": tool.get_url(),
            "last_published_at": tool.last_published_at.isoformat() if tool.last_published_at else None,
        })

    return Response({
        "tools": tools_data,
        "pagination": {
            "page": page,
            "size": size,
            "total": queryset.count(),
            "has_next": end < queryset.count(),
            "has_prev": page > 1
        },
        "categories": {}, # 分类统计
        "filters": {
            "category": category,
            "search": search,
            "pricing": pricing,
            "is_hot": is_hot,
            "is_new": is_new
        }
    })
```

```typescript
            </span>
          </div>
          <span
            className={`text-xs px-2 py-1 rounded-full ${
              tool.pricing === "free"
                ? "bg-green-100 text-green-800"
                : tool.pricing === "freemium"
                ? "bg-blue-100 text-blue-800"
                : "bg-orange-100 text-orange-800"
            }`}

              <div className="flex items-center space-x-1">

              </div>

                className={`text-xs px-2 py-1 rounded-full ${

                    ? "bg-blue-100 text-blue-800"

              >

                  : tool.pricing === "freemium"
                  ? "免费版"
                  : tool.pricing === "paid"
                  ? "付费"
                  : "企业版"}
              </span>
            </div>

          <div className="flex items-center justify-between">
            <div className="flex flex-wrap gap-1">
              {tool.tags.slice(0, 3).map((tag) => (
                <span
                  key={tag}
                  className="text-xs bg-gray-100 text-gray-600 px-2 py-1 rounded-full"
                >
                  {tag}
                </span>
              ))}
            </div>

            <button className="text-blue-600 hover:text-blue-700 text-sm font-medium flex items-center space-x-1">
              <span>访问</span>
              <ExternalLink className="w-3 h-3" />
            </button>
          </div>
        </div>
      </>
    )}
  </motion.div>
);
```

## 常见问题和解决方案

### 1. 容器启动问题

**问题**: 容器启动失败或端口冲突

```bash
# 解决方案：清理容器和重新构建
docker compose -f infra/local/docker-compose.yaml down -v
docker compose -f infra/local/docker-compose.yaml up -d --build --force-recreate
```

### 2. 数据库迁移问题

**问题**: 迁移文件冲突或字段不匹配

```bash
# 解决方案：重置迁移状态
docker compose -f infra/local/docker-compose.yaml exec authoring python authoring/manage.py migrate app_name zero --fake
docker compose -f infra/local/docker-compose.yaml exec authoring python authoring/manage.py makemigrations
docker compose -f infra/local/docker-compose.yaml exec authoring python authoring/manage.py migrate
```

### 3. 前端依赖问题

**问题**: npm 包版本冲突或安装失败

```bash
# 解决方案：清理并重新安装依赖
docker compose -f infra/local/docker-compose.yaml exec portal rm -rf node_modules package-lock.json
docker compose -f infra/local/docker-compose.yaml exec portal npm install
```

### 4. Tailwind CSS 构建问题

**问题**: CSS 样式不生效或构建失败

```bash
# 解决方案：检查配置并重新构建
docker compose -f infra/local/docker-compose.yaml exec portal npm run build
# 或者重启开发服务器
docker compose -f infra/local/docker-compose.yaml restart portal
```

### 5. API CORS 问题

**问题**: 前端无法访问后端 API

```python
# 解决方案：在Django设置中添加CORS配置
CORS_ALLOW_ALL_ORIGINS = True  # 仅开发环境
CORS_ALLOWED_ORIGINS = [
```
